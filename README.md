Smart Travel Companion is a Java-based desktop application designed to act as a complete travel guide by centralizing essential travel information into a single platform. The application helps users plan trips efficiently by providing real-time data, safety insights, and itinerary management features.

## Features

* City and state-based travel information
* Real-time weather and air quality updates
* Safety and crime statistics
* Tourist attractions, hotels, restaurants, and shopping centers
* Live news with category filtering
* Tour planner with route and place suggestions
* Trip management (My Trips) with itinerary creation

## Technologies Used

* Java (Java 8+)
* Java Swing (UI)
* External APIs (Weather, News, Routing)
* org.json for JSON parsing

## How to Run

1. Clone the repository
2. Open the project in any Java IDE (NetBeans / IntelliJ / Eclipse)
3. Configure API keys if required
4. Run the `SmartTravelCompanion` main class

## Project Purpose

This project was developed to solve the problem of fragmented travel information by providing a unified, user-friendly desktop application for travel planning and decision-making.
